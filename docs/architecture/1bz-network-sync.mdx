## Associate Registry

### SQLite Table

```sql
CREATE TABLE associates (
    id INTEGER PRIMARY KEY,
    network_id TEXT UNIQUE,
    name TEXT,
    telegram_id TEXT,
    email TEXT,
    role TEXT,
    business_type TEXT,
    status TEXT DEFAULT 'active'
);
```

### Associate Attributes

| Field | Description |
|-------|-------------|
| `network_id` | Unique 1bz network identifier |
| `name` | Associate's business name |
| `telegram_id` | Telegram contact |
| `email` | Email address |
| `role` | Role (logistics, marketing, legal, etc.) |
| `business_type` | Industry category |
| `status` | active/inactive |

## Network ID Model

### ID Format

```
1bz:associate:ABC123XYZ
```

Components:
- `1bz` - Network prefix
- `associate` - Type
- `ABC123XYZ` - Unique hash

## Metadata Hashing

### Hash Generation

```python
import hashlib
import json

def hash_metadata(metadata):
    metadata_str = json.dumps(metadata, sort_keys=True)
    return hashlib.sha256(metadata_str.encode()).hexdigest()
```

### Sync Record

```sql
CREATE TABLE sync_registry (
    id INTEGER PRIMARY KEY,
```
