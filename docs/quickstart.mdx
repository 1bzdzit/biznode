## Step 1: Clone the Repository

```bash
git clone https://github.com/your-repo/biznode.git
cd biznode
```

---

## Step 2: Configure Environment Variables

Copy the example environment file:

```bash
cp .env.example .env
```

Edit `.env` with your settings:

```bash
# Required: Telegram Bot Token
TELEGRAM_BOT_TOKEN=your_bot_token_here

# Required: Your Telegram User ID
OWNER_TELEGRAM_ID=your_telegram_id

# Optional: Email Settings (for Gmail use app password)
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=your_email@gmail.com
SMTP_PASSWORD=your_app_password
AGENT_EMAIL=ai@yourbiznode.1bz
```

---

## Step 3: Set Up Telegram Bot

1. Open Telegram and search for **@BotFather**
2. Send `/newbot` to create a new bot
3. Follow the prompts to name your bot
4. Copy the **API Token** provided
5. Add the token to your `.env` file

```bash
TELEGRAM_BOT_TOKEN=1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
```

## Step 4: Set Up AI Email (Optional)

---

## Step 5: Start Docker Compose

```bash
docker compose up -d
```

This will start:
- **Ollama** - Local LLM (Qwen2.5)
- **Qdrant** - Vector database
- **BizNode Agent** - Main application

Check status:

```bash
docker compose ps
```

Expected output:

```
NAME                IMAGE               STATUS
biznode-ollama      ollama/ollama       Up
biznode-qdrant      qdrant/qdrant       Up
biznode             biznode/app         Up
```

---

## Step 6: Initialize the Database

```bash
docker compose exec biznode python -c "from memory.database import init_db; init_db()"
```

---

## Step 7: Configure Agent Identity

Create a Python script to set up your AI agent:

```python
from memory.database import save_agent_identity

identity = {
    "agent_name": "MyBizNode",
    "agent_email": "ai@mybiznode.1bz",
    "telegram_bot_token": "YOUR_BOT_TOKEN",
    "owner_telegram_id": "YOUR_TELEGRAM_ID",
    "owner_email": "you@example.com",
    "autonomy_level": 1  # 1=assistive, 2=semi-auto, 3=full
}
save_agent_identity(identity)
```

Run it:

```bash
docker compose exec biznode python setup_identity.py
```

## Troubleshooting

### Docker Not Starting

```bash
# Check Docker status
docker info

# Restart Docker
# (Windows) Restart Docker Desktop
# (Linux) sudo systemctl restart docker
```

### Ollama Not Responding

```bash
# Pull the model explicitly
docker compose exec ollama ollama pull qwen2.5

# Check logs
docker compose logs ollama
```

### Qdrant Connection Error

```bash
# Check if Qdrant is running
docker compose ps

# Restart Qdrant
docker compose restart qdrant

# Check logs
docker compose logs qdrant
```

### Telegram Bot Not Responding

1. Verify bot token is correct in `.env`
2. Check the bot has been started (/start in Telegram)
3. Review logs: `docker compose logs biznode`
