# Telegram Bot Setup

This guide walks through setting up a Telegram bot for BizNode.

---

## Step 1: Create a Bot

1. Open Telegram
2. Search for **@BotFather**
3. Send `/newbot`
4. Follow the prompts:
   - Bot name: `MyBizNode`
   - Bot username: `@mybiznode_bot`
5. Copy the **API Token** (e.g., `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`)

---

## Step 2: Configure Environment

Add the token to `.env`:

```bash
TELEGRAM_BOT_TOKEN=1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
```

---

## Step 3: Get Your Telegram ID

### Option A: @userinfobot

1. Search for **@userinfobot** in Telegram
2. Start the bot
3. It will show your user ID

### Option B: @myidbot

1. Search for **@myidbot**
2. Send `/getid`
3. Copy your user ID

---

## Step 4: Set Owner ID

Add to `.env`:

```bash
OWNER_TELEGRAM_ID=123456789
```

---

## Step 5: Test the Bot

1. Start your bot: `/start`
2. Check status: `/status`

---

## Bot Commands

| Command | Description |
|---------|-------------|
| `/start` | Initialize bot |
| `/status` | Check system health |
| `/help` | Show help |
| `/register [name]` | Register business |
| `/search [query]` | Search memory |

## Webhook vs Polling

### Polling (Default)

```python
# In telegram_service.py
from telegram.ext import Application, MessageHandler, CommandHandler

app = Application.builder().token(TOKEN).build()
app.run_polling()
```

### Webhook (Production)

```bash
# Set webhook
export WEBHOOK_URL=https://your-domain.com/webhook
```

```python
# Set webhook
app = Application.builder().token(TOKEN).build()
app.run_webhook(
    listen="0.0.0.0",
    port=443,
    url_path="webhook",
    webhook_url="https://your-domain.com/webhook"
)
```

## Security

### Bot Privacy

1. Go to @BotFather
2. Select your bot
3. Choose **Bot Settings** → **Privacy Mode**
4. Enable privacy mode

### Group Permissions

If using in groups:
1. @BotFather → Bot Settings → Group Privacy
2. Turn off to allow group commands
